# Bootstrap

## 布局容器

![image-20220709112220722](readme.assets/image-20220709112220722.png)



Bootstrap的流式栅格系统，随着屏幕或视图(viewport)尺寸的增加，系统会自动分为最多**12列**；



添加了`container`类后，元素和左边有padding值，添加`row`类名后，可以消除padding值。
